# Goal

## What
Develop cog-focus — a cognitive architecture plugin that gives Claude persistent, goal-oriented memory across sessions.

## Why
- Claude can drive longer projects while keeping consistency in design and decisions
- Past pitfalls for a given project are remembered and avoided
- Sessions build on each other instead of starting from scratch

## Success Looks Like
- Evolving, open-ended — this project grows with use and may be dropped anytime
- No fixed finish line; success is measured by whether it's actually useful in practice

## Constraints
- None — side project, no deadlines, scope evolves organically
